# Released Checkpoints

This branch includes only checkpoints that fit normal GitHub hosting without LFS:

- `trnet_seed44_best.pt`
  Paper trajectory baseline used in the released comparison table.

- `trajectory_only_legacy_best.pt`
  Earlier lightweight trajectory-only checkpoint kept for compatibility and quick loading tests.

Large image, audio, and trimodal weights are intentionally omitted from this branch because they exceed typical GitHub file-size limits.
