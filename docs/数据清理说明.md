# 清理无意义数据列

## 修改日期
2026-03-05

## 问题描述

在 `data/aligned_output/aligned_data.csv` 文件中，存在一些无意义的数据列，这些列对模型训练和分析没有帮助：

1. **上点时间** - GPS定位的辅助时间信息
2. **补点** - GPS补点标记
3. **unitid** - 设备单元ID
4. **GPS标记** - GPS定位标记
5. **播种播肥 / 油耗 / 压力** - 多个设备参数混在一起
6. **抛肥量(立方)** - 大部分为0或空值
7. **定位间隔** - GPS定位间隔

## 解决方案

### 1. 创建清理脚本

创建了 `scripts/clean_aligned_data.py`，用于清理现有的CSV文件。

**功能**:
- 读取 aligned_data.csv
- 删除指定的无意义列
- 保存清理后的数据

**使用方法**:
```bash
python scripts/clean_aligned_data.py --input data/aligned_output/aligned_data.csv
```

### 2. 修改对齐脚本

修改了 `scripts/align_video_trajectory_final.py`，在加载数据时自动过滤这些列。

**修改内容**:
- 添加全局常量 `COLUMNS_TO_REMOVE`
- 在 `__init__` 方法中加载数据后自动过滤这些列
- 输出过滤的列数统计

## 清理结果

### 原始数据
- **总列数**: 22
- **总行数**: 6272

### 清理后数据
- **总列数**: 15 (删除了7列)
- **总行数**: 6272 (保持不变)

### 保留的列

1. `frame_path` - 帧图片路径
2. `frame_time` - 帧时间戳
3. `video_file` - 视频文件名
4. `frame_number` - 帧序号
5. `second_in_video` - 视频中的秒数
6. **定位时间** - GPS定位时间
7. **经度** - GPS经度
8. **纬度** - GPS纬度
9. **间距(米)** - 与前一点的距离
10. **深度** - 耕作深度
11. **速度** - 移动速度
12. **类型** - 操作类型
13. **方向角** - 移动方向
14. **分类** - 农业活动分类标签
15. **时间戳** - Unix时间戳

### 删除的列

1. ~~上点时间~~
2. ~~补点~~
3. ~~unitid~~
4. ~~GPS标记~~
5. ~~播种播肥 / 油耗 / 压力~~
6. ~~抛肥量(立方)~~
7. ~~定位间隔~~
