# 时间偏移更新说明

## 更新日期
2026-03-05

## 更新内容
根据用户反馈，视频截图和对应名字大部分相差1秒，因此将所有视频帧的时间戳**增加1秒**。

## 修改详情

### 代码修改
文件: `scripts/align_video_trajectory_final.py`

```python
# 修改前
frame_time = start_time + timedelta(seconds=second)

# 修改后
frame_time = start_time + timedelta(seconds=second + 1)  # 增加1秒偏移
```

### 时间变化

#### 第一个视频 (20241018043810.mp4)
- **修改前**: 12:38:12 开始
- **修改后**: **12:38:13** 开始
- **偏移**: +1秒

#### 最后一个视频 (20241018111131.mp4)
- **修改前**: 19:41:29 结束
- **修改后**: **19:41:30** 结束
- **偏移**: +1秒
