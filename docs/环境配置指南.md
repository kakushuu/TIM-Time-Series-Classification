# Agri-MBT 环境配置指南

## 📋 环境说明

**环境名称**: `agri-mbt`
**Python 版本**: 3.10
**PyTorch 版本**: 2.0+ (CUDA 11.8)

### 主要依赖

- **深度学习框架**: PyTorch 2.0+, torchvision, torchaudio
- **视觉处理**: OpenCV, Pillow
- **OCR**: pytesseract
- **模型库**: timm (PyTorch Image Models), transformers
- **数据处理**: numpy, pandas, scipy, scikit-learn
- **可视化**: matplotlib, seaborn
- **工具**: tqdm, openpyxl

---

## 🚀 快速开始

### 方法 1: 使用自动脚本 (推荐)

```bash
# 给脚本添加执行权限
chmod +x setup_env.sh

# 运行脚本
./setup_env.sh

# 激活环境
conda activate agri-mbt
```

### 方法 2: 手动安装

```bash
# 1. 创建基础环境
conda create -n agri-mbt python=3.10 -y

# 2. 激活环境
conda activate agri-mbt

# 3. 安装 PyTorch (CUDA 11.8)
pip install torch==2.0.1+cu118 torchvision==0.15.2+cu118 torchaudio==2.0.2+cu118 \
    --index-url https://download.pytorch.org/whl/cu118

# 4. 安装核心依赖
pip install numpy scipy pandas scikit-learn
pip install matplotlib seaborn tqdm
pip install opencv-python pillow pytesseract
pip install openpyxl

# 5. 安装深度学习库
pip install timm transformers einops

# 6. 安装 Jupyter (可选)
pip install jupyter ipywidgets
```

### 方法 3: 使用 requirements.txt

```bash
# 创建环境
conda create -n agri-mbt python=3.10 -y
conda activate agri-mbt

# 安装依赖
pip install -r requirements.txt
```

## 🐛 常见问题

### Q1: CUDA 不可用?

**检查**:
```bash
nvidia-smi  # 检查 GPU 状态
nvcc --version  # 检查 CUDA 版本
```

**解决**:
- 确保 NVIDIA 驱动已安装
- 确保安装的是 CUDA 版本的 PyTorch
- 检查 CUDA 版本是否匹配 (本环境使用 CUDA 11.8)

### Q2: 导入错误?

**检查**:
```bash
# 确认在正确的环境中
conda info --envs
which python  # Linux/Mac
where python  # Windows
```

**解决**:
```bash
conda activate agri-mbt
```

### Q3: 包版本冲突?

**解决**:
```bash
# 方法1: 升级包
pip install --upgrade package_name

# 方法2: 重新安装
pip uninstall package_name
pip install package_name

# 方法3: 重建环境 (最后手段)
conda env remove -n agri-mbt
./setup_env.sh
```

### VS Code

1. 安装 Python 扩展
2. 选择 `agri-mbt` 环境:
   - `Ctrl+Shift+P` (Windows/Linux) 或 `Cmd+Shift+P` (Mac)
   - 输入 "Python: Select Interpreter"
   - 选择 `agri-mbt` 环境

### PyCharm

1. Settings > Project > Python Interpreter
2. Add Interpreter > Conda Environment
3. 选择 `agri-mbt`
